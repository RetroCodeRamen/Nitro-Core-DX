# Known Issues and Fixes

This directory contains issue investigations, fix writeups, and troubleshooting notes.

## How To Use This Folder

- Treat these files as **incident history / debugging references**, not a current status dashboard.
- Many files describe specific bugs that are already fixed.
- For current status, use:
  - `README.md`
  - `docs/HARDWARE_FEATURES_STATUS.md`
  - `docs/testing/TEST_SUMMARY.md`
  - recent test runs/worklog entries

## Files

- **BLACK_SCREEN_FIX.md** - Fix for blank screen issues
- **CGRAM_WRITE_FIX.md** - Fix for CGRAM write issues
- **DEBUG_INFINITE_LOOP.md** - Fix for infinite loop debugging issues
- **ROM_COMPATIBILITY_FIX.md** - ROM compatibility fixes
- **VBLANK_FIX_SUMMARY.md** - VBlank synchronization fixes
- **VBLANK_FLAG_ANALYSIS.md** - VBlank flag analysis
- **CPU_EXECUTION_VERIFICATION.md** - CPU execution verification
- **INPUT_FIX_DEBUG.md** - Input system debugging and fixes
